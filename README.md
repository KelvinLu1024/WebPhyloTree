# WebTree
A Web-based embedable dynamic tree viewer. Translate JSON into SVG. Lightweight and extensible.

# Feature
* Layouts
  * Rectangular
  * Circular
  * Unrooted
* Appendages
  * label
  * labelButton
  * Button
* Extensions
  * nodeFold
  * leafSelection
  * ... (coming soon)
  
# Requirements
__None__

# Usage
## HTML
'''<script src="js/webtree.js"></script>'''

## XML
'''<script xlink:href="svg-helper.js" type="text/javascript" />'''
